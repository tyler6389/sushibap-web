# 🍚 밥물 계산기

주방에서 바로 쓰는 **쌀–물 비율 계산 웹앱**입니다.  
씻기 전/후 쌀 무게만 입력하면 **실제로 넣어야 할 물 양(g/ml)**을 자동으로 계산해줍니다.

---

## ✨ 기능

- 씻기 전 쌀 무게 입력
- 씻은 후 쌀 무게 입력
- 쌀이 이미 머금은 물을 제외하고  
  👉 **추가로 넣어야 할 물 양 자동 계산**
- 기준 비율: **쌀 : 물 = 1 : 1**
- **모바일 최적화**
- iPhone **홈 화면에 앱처럼 추가 가능**

---

## 📐 계산 방식

1. 필요 물 = 씻기 전 쌀 무게  
2. 머금은 물 = 씻은 후 무게 − 씻기 전 무게  
3. 실제 넣을 물 = 필요 물 − 머금은 물
